title: Keeping things updated
date: 25 May 2017
---

- bu 
	- aliased command in .bash_profile
    - Updates homebrew, including cask
- upall
	- aliased command in .bash_profile
	- performs updates for brew, pip, npm, gem
- pipUp
    - aliased command in .bash_profile
- npm 
	- npm -v
	- npm install -g npm [installs globally]
	- npm outdated -g
	- npm update -g <pkg>
- gem
	- gem update
